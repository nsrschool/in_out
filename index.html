📖 คู่มือการใช้งานอย่างย่อ

  1. ตั้งค่าระบบครั้งแรก: ไปที่ ⚙️ ตั้งค่าระบบ → กรอกข้อมูลโรงเรียน อัปโหลดโลโก้และลายเซ็น ผอ. → กดบันทึก (โลโก้จะกลายเป็นไอคอนของเว็บอัตโนมัติ)
  2. เพิ่มรายชื่อนักเรียน: แท็บ 🧑‍🎓 จัดการรายชื่อนักเรียน → นำเข้าไฟล์ Excel/CSV หรือเพิ่มทีละคน
  3. รับสมัคร/ย้ายเข้า: กรอกฟอร์มตามใบสมัคร ติ๊กหลักฐาน → พิมพ์ใบสมัคร/แบบตอบรับ → เมื่อกด "รับเข้าเรียน" ระบบเพิ่มชื่อลงทะเบียนนักเรียนให้อัตโนมัติ
  4. ย้ายออก: บันทึกคำร้อง → ตรวจหลักฐาน → กด "ออกหนังสือนำส่ง" ระบบรันเลขที่หนังสือให้อัตโนมัติ → พิมพ์คำร้อง/หนังสือนำส่ง
  5. ปพ.1 / ปพ.7: บันทึกการออกเอกสารลงทะเบียนคุม — ปพ.7 พิมพ์ฉบับจริงได้จากระบบ
  6. สำรองข้อมูล: กดปุ่ม 💾 ด้านบนเพื่อดาวน์โหลดไฟล์ .json เก็บไว้ และนำกลับมา 📥 นำเข้าได้ทุกเมื่อ

🚀 วิธี Deploy เป็น Web App (สำหรับครูที่นำไปใช้ต่อ)

  1. สร้าง Google Sheet ใหม่ → เมนู Extensions → Apps Script
  2. วางโค้ด Code.gs ลงไป แล้วกดบันทึก
  3. กด Deploy → New deployment → Web app → Execute as: Me → Who has access: Anyone → Deploy (อนุญาตสิทธิ์ครั้งแรกตามที่ระบบถาม)
  4. คัดลอก Web App URL มาวางในไฟล์ index.html บรรทัดบนสุด (ตัวแปร WEB_APP_URL)
  5. อัปโหลด index.html ขึ้น GitHub Pages → เปิดใช้งานผ่านลิงก์ของ GitHub Pages ได้ทันที
  6. ⚠️ ทุกครั้งที่แก้ Code.gs ต้อง Deploy → Manage deployments → ✏️ Edit → New version → Deploy / แก้ index.html ต้องอัปโหลดขึ้น GitHub ใหม่แล้ว Hard Refresh (Ctrl+Shift+R)
